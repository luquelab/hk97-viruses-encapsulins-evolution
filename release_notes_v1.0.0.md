# Code accompanying the accepted article (v1.0.0)

This is the archival code release associated with the accepted Nature Communications article:

**Small viruses reveal bidirectional evolution between HK97-fold viruses and encapsulins via procapsids**

The release contains:

- self-contained workflows and compact inputs for the domain co-occurrence network and Fig. 3 domain-coverage analyses;
- the statistical workflow and unrounded pairwise outputs used in the manuscript;
- interactive R Shiny and Python Tk applications used to generate representative visualizations;
- supporting upstream utilities retained for provenance; and
- links to the complete datasets deposited in Figshare.

See `README.md` for usage, `VALIDATION.md` for release checks, and `bin/utils/README.md` for the scope and external requirements of upstream utilities.
